仁愛慷慨

子產

簡介 : 春秋時期鄭國政治家，思想家， 在位期間推行一系列改革， 為人仁愛寬厚， 經常濟貧救，危深受人民愛戴。

  • 事例 : 有客人贈送他活魚(食用珍品)， 他把魚放在魚池中飼養， 他說「為了口腹之慾而把魚活生生殺死，我於心不忍。」 可見他仁心推廣至一般生物上。
  • 延伸 : 鄭國人經常在鄉校議論國政，有人建議把拆毀，防止滋生事端，影響社會穩定。子產認為他們是他的老師，可以糾正他的施政錯誤。可見他尊重民意，有為官之道。

喬安娜羅琳

簡介 : 著名科幻小說《哈利波特》的作者，婚姻失敗後帶著女兒堅強地生活， 最後寫出著名小說，名利雙收， 可是她沒有被名利蒙蔽雙眼， 更盡己所能從事慈善工作。

  • 事例 : 她曾把《哈利波特》的版稅收入捐給英國喜劇救濟會附設的「哈利波特基金」， 專門幫助一些有需要的兒童，讓這些兒童能過上正常的童年，見出其仁愛慷慨。
  • 延伸 : 離婚後她一直找不到工作只能靠微薄的失業救濟金過活，但她沒有自暴自棄，反而更加努力， 以一台老舊的人工打字機完成了《哈利波特》第一集， 可見她的積極奮發和堅毅不屈。

周潤發

簡介 : 香港知名男演員，演藝成就備受肯定，然而他沒有沉醉在名利中，一直貫徹「取之社會，用之社會」的理念行事，非常熱心慈善。

  • 事例 : 周潤發效法巴菲特、比爾蓋茨等美國富翁，決定死後把財產的九十九%捐給慈善機構，讓他們能夠用這筆錢去幫助更多有需要的人，可見他的無私與慷慨。
  • 延伸 : 人在中年時代才學習是非常困難的， 周潤發四十歲才學習英語， 但他不怕辛苦，付出雙倍的努力， 每天苦練英語兩小時， 強迫自己看英語節目。 最後終於能懂英語，成為美國荷里活的國際巨星， 可見他奮發堅毅的好學精神。

樂觀積極

項羽

簡介 : 項羽在秦末率領民眾起兵反秦， 起兵三年後，他已率領六國諸侯殲滅秦國，並分封天下。

  • 事例 : 楚漢戰爭後期， 項羽在垓下之戰被劉邦統領的六十萬聯軍擊敗。 突圍至烏江江邊後， 烏江亭長對項羽說他已準備了船讓他渡江，只要他乘上船就不會被漢軍追上。但項羽卻以「無顏見江東父老」為由拒絕，自刎而死， 可見他不能「包羞忍恥」。
  • 延伸 : 項羽年少時， 有次與叔父觀看秦始皇出巡會稽的行列， 項羽看見聲勢浩大的儀仗隊， 忍不住說「彼可取而代之」，可見他自小已有非凡的志氣。

余翠怡

簡介 : 香港輪椅劍擊運動員， 11歲時罹患骨癌，必須切除左腿小腿來保命。 然而她截肢之後沒有失去人生方向。

  • 事例 : 失去左腿小腿的余翠怡沒有心灰意冷， 反而樂觀地接受現實，甚至安慰身邊為她傷心難過的家人。
  • 延伸 : 獲得眾多國際比賽金牌後， 余翠怡沒有因而自滿，反而繼續勤奮的練習劍擊， 備戰其他國際賽事， 並立志在巴西殘奧再為港爭光， 雖然無法成功衛冕金牌，但其遠大的理想和堅毅不屈的意志贏得香港人的熱烈掌聲。

霍金

簡介 : 英國物理學家， 他因病全身癱瘓，不能發聲， 但他仍然醉心研究，希望徹底了解宇宙， 在物理學上貢獻良多。

  • 事例 : 霍金最初被診斷出患有致命的疾病後， 完全失去求生的意志，在老師的鼓勵下，他重拾已放棄的學業， 並研究室著名的黑洞理論， 他的成就源自於他的堅毅不屈。
  • 延伸 : 霍金雖然一直不願被視為殘疾的人， 但亦呼籲政府設定預防殘疾措施， 保障殘障人士權利， 致力為殘障人士爭取權益， 可見他的仁愛慷慨。

知錯能改

李白

簡介 : 中國唐朝詩人， 他的作品以想像雄有豐富、浪漫奔放、 意境獨特著稱。 他的詩流傳千古，為中國文學帶來極大影響。

  • 事例 : 李白小時候不喜歡讀書， 一次逃學時遇到一個老婦正在磨鐵杵， 他問老婦在做什麼， 她回答說要把鐵杵磨成針，只要每天磨，工夫不負有心人，一定能幫它磨成針。 李白深受感動，明白學習也應該有這樣的恆心，於是回去努力讀書成為名垂千古的大詩人。
  • 延伸 : 李白棄官後， 在十年間足跡遍及多個地方， 其他寫下不少描繪自然風景的詩篇， 表現了他的豪情壯志和開闊胸襟， 反映他追求不平凡事物的渴求。

華盛頓

簡介 : 華盛頓是美國第一任總統， 他在美國獨立戰爭時擔任殖民地軍的總司令與英軍對戰， 為戰爭定立正確的戰略，直到英軍承認美國獨立。 因而被稱為美國國父。

  • 事例 : 華盛頓小時候想測試他喜歡的小斧頭的威力， 於是砍掉了家中十分珍貴的櫻桃樹。 爸爸發現後很生氣， 但是華盛頓沒有逃避責任，後來爸爸不但沒有責怪他，反而讚賞他勇於認錯。
  • 延伸 : 華盛頓在道德和經濟上均反對奴隸制， 他立遺囑釋放所有奴隸， 不容許自己的繼承人購買或轉送奴隸， 並使他們得到適切的安置， 可見他對奴隸的仁愛之心，沒有因個人地位而任意剝削他人應有的權利。
