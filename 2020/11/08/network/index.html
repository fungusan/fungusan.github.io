Definition

  • Depends on how large they are and how much of an area they cover geographically.

PAN

  • Stands for Personal Area Network.
  • Used on a personal level.
  • Connects devices such as smartphones, tablets and laptops by using wireless technologies (Bluetooth, infrared, NFC).
  • But also can be connected by using a wired connection (USB cable).
  • Generally used for transferring small files.

LAN

  • Stands for Local Area Network.
  • A group of devices that are located in the same building.

Ethernet LAN is the most common type of LAN : Two or more computers are connected to Ethernet cables using a switch.

Peer-to-peer (2P2) network

Interconnected nodes (“peers”) share resources amongst each other without the use of a centralized administrative system

  • P2P network is a distributed network that stores and transfers data without a central server.
  • Peers are equally privileged, equipotent participants in the application.

Advantages

  • No need for a network operating system.
  • Does not need an expensive server because individual workstations are used to access the files
  • Much easier to set up than a client-server network

Disadvantages

  • Difficult to backup the data as it is stored in different computer systems and there is no central server.
  • Hard to provide overall security in the peer to peer network as each system is independent and contains its own data.
  • Because each computer might be being accessed by others it can slow down the performance for the user.

Client-Server Network

Clients communicating with a server via the Internet

  • Use a dedicated computer (server) to store data, manage/provide resources and control user access.
  • The server acts as a central point on the network upon which the other computers connect to.
  • A computer that connects to the server is called a client.

Advantages

  • Facilitate system backup and improve fault tolerance.
  • Facilitate resource sharing - Centrally administer and control.
  • Enhance security - Only administrator can access to the server.
  • Supports more users

Disadvantages

  • More expensive to configure their hardware and software components
  • To need well qualified technical staff for maintaining the server.
  • Client server network is need special network operating system.

WLAN

  • Stands for Wireless Local Area Network.
  • Use wireless communication.
  • A WLAN is a LAN but without using cables.

CAN

  • Stands for Campus Area Network.
  • Joins two or more Lans together within a limited area.

MAN

  • Stands for Metropolitan Area Network.
  • Spans over several buildings in a city or town.

SAN

  • Stands for Storage Area Network.
  • Used to access storage devices, such as disk arrays and tape libraries from servers so that the devices appear to the operating system as direct-attached storage.
  • A special high-speed network that stores and provides access to large amounts of data.
  • Not affected by network traffic.

WAN

  • Stands for Wide Area Network.
  • Largest type of network
  • Spans over a large geographical area.

The internet is an example of a WAN.
