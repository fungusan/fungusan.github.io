Network Hardware

Network Interface Controller/Card (NIC)

  • A device need a NIC to be able to connect to a network.
  • All NICs have a unique identifier called a Media Access Control(MAC) Address.

Media Access Control(MAC) Address

  • Every device on the network has its own MAC address,unique worldwide.
  • MAC is hard-coded into the NIC, so can’t be changed.

Switch

https://media.4rgos.it/i/Argos/8663236_R_Z001A?w=1500&h=880&qlt=70&fmt=webp

  • It has lots of ports to connect devices in a LAN.
  • It accepts Ethernet connections from Network Devices.
  • It forwards packets to devices in other segments of the LAN.
  • When a data packet is sent to a switch, it’s only directed to the intended destination port as it stores MAC addresses in its table.

Data sent over a network is broken down into small network packets.

Hub

  • It accepts Ethernet connections from Network Devices.
  • It doesn’t filter any data as it only knows when a device is connected to on of its ports.
  • When a data packet arrives at one of the ports, it is copied to all of the other ports.

Disadvantages

  • Creates security concerns.
  • Creates unnecessary traffic on the Network. -> (wastes banwidth)

Router

  • It routes or forwards data from one network to another based on their IP address.
  • When a data packet is received, It inspects the data’s IP address and determines if the packet was meant from its own network.

If so, the router receives it. If not, the router sends it off to another network.

  • It is the gateway of a network.
  • It routes the internet connection to all devices

Modem

  • It brings the internet to home or business.
  • It establishes and maintains a dedicated connection to ISP(Internet Service Provider) to give access to internet.

A computer only reads digital signal. Internet only reads analog signal.

  • As analog signal come in from internet, it demodulates the analog signals in digital signals. Vice versa.
